Cos'è il Web?

"Il Web è progettato [...] per essere universale:
per includere tutto e tutti."
Tim Barners-Lee

Il Web connette le persone

Il protocollo HTTP

permette di far comunicare facilmente client e server
usando dei messaggi di testo


HTML

il linguaggio di Markup del Web

usa i tag <> per creare blocchi di contenuto


1989 CERN, Ginevra

Tim Barners-Lee

presenta un documento in cui propone un nuovo modo per gestire le informazioni


"Il Web è più un'innovazione sociale che un'innovazione tecnica. L'ho progettato perché avesse una ricaduta sociale, perché aiutasse le persone a collaborare, e non come un giocattolo tecnologico. Il fine ultimo del Web è migliorare la nostra esistenza reticolare nel mondo."

Tim Barners-Lee

Il Web è uno Standard

Il W3C, fondato da Tim Barners-Lee, si occupa di definire lo standard web e di mantenerlo libero

Il Web è la piattaforma del FOSS

Free and Open Source Software

0. Libertà di Utilizzare

Posso usare il software come voglio, per ottenere quello che voglio

1. Libertà di Analizzare

Posso prendere il codice sorgente e studiarlo, per imparare o per controllarne la qualità.

2. Libertà di Modificare

Posso riscrivere il software, implementare nuove funzionalità o correggere errori.

3. Libertà di Ridistribuire

Sono libero di condividere quello che ho realizzato, ho la libertà di "copiare cambiare qualche parola e consegnare il compito".

Web 1.0

È statico e ha poca interazione con l'utente

L'utente può cercare e leggere i contenuti

È una raccolta mondiale di contenuti multimediali

Il più grande libro collaborativo dell'umanità

Non c'erano grandi innovazioni, ma i contenuti erano liberi

Aveva capacità molto limitate

Non era assolutamente bello...

Web 2.0

È dinamico e interattivo

Basta refresh (F5)

Le persone sono in grado di interagire tra di loro

Meno elitario, chiunque può creare contenuti

Non è più un solo libro, ma diventa una vera e propria Realtà Aumentata e Virtuale

Le nuove tecnologie permettono di fare del Web la piattaforma per (quasi) ogni cosa

Il Web arriva sui dispositivi Mobile: chiunque è connesso sempre, ovunque sia
