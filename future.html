Tecnologie Web Rivoluzionarie

1996

Le tecnologie web non sono abbastanza potenti, Macromedia introduce Flash, un software distribuito tramite plugin che mette una pezza alle mancanze del web.
Negli anni diventa uno standard de facto


2012 - 2014

Il W3C standardizza HTML5 come nuovo linguaggio per il web, introducendo il supporto nativo a audio, video, canvas e altre tecnologie. Anche Youtube si converte a HTML5
